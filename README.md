## Learning Approach

For **every topic**:
1. Start with the **basic files** to understand the concept.
2. Move to the **mini demo** to see how the same concept works in practice.
3. Code is intentionally kept **simple and readable**.
4. Comments are added to explain *why* something is done, not just *what* is done.
